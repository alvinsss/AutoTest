
java版的apitest介绍<br>
=================<br>

1、目录结构
-----------
cases：具体的测试用例编写<br>
constants:静态变量及测试数据xlsx路径<br>
pojo:使用easypoi对excel列和实体类的成员变量映射关系<br>
utils:对Excel http请求 jdbc等封装<br>

2、使用说明
-----------

1、 通过指定testng文件及内容中的类和方法确定运行cases(jenkins构建之后执行此文件)
